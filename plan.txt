   - `python scripts/ci_cleanup.py --root . --check`  
   - `npm.cmd --prefix apps/web run lint`  
   - `npm.cmd --prefix apps/web run test`
2. **Avant push / merge (moyen, ~2-6 min)**  
   - `npm.cmd --prefix apps/web run build`  
   - `powershell -ExecutionPolicy Bypass -File scripts/run_checks.ps1 -Scope changed -SkipE2E -SkipEncodingAutofix`
3. **Gate pré-release uniquement**  
   - `powershell -ExecutionPolicy Bypass -File scripts/run_checks.ps1 -Scope full`
