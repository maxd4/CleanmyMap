# Software Wiki

This wiki is the operational documentation space for both:
- end users (what changed, how to use it),
- developers (implementation impact, files touched, tests, migration notes).

## Update Rule

Any significant update must be reflected in both:
- `README.md` (high-level and user-facing summary),
- this wiki (detailed implementation and maintenance notes).

If one is missing, the change is considered undocumented.
