# Documentation du Master Pack (Rapport d'Impact Exhaustif)

## 1. Vision
Le Master Pack est le standard institutionnel de CleanMyMap pour la restitution de l'impact territorial. Il est conçu pour être **impartial**, **transparent** et **actionnable**.

## 2. Structure du Rapport

### 2.1 En-tête & Certification
- **ID de Livrable** : Identifiant unique pour la traçabilité.
- **Data Seal** : Badge de certification "Protocol v3.0".
- **Périmètre** : Précision du filtrage (Global / Association / Entreprise).

### 2.2 Synthèse Exécutive (Décideur)
- **Score de Confiance (Readiness)** : Moyenne pondérée de la complétude, cohérence et géolocalisation.
- **Headline** : Résultat majeur de la période.
- **Budget Use Cases** : Recommandations d'arbitrage financier basées sur la donnée.

### 2.3 Section Opérationnelle (Terrain)
- **Indice de Récurrence** : Mesure du retour des déchets sur une zone nettoyée.
- **Couverture Cartographique** : % de zones couvertes par des preuves spatiales (traces, polygones).
- **Itinéraires Recommandés** : Algorithme de priorité basé sur les spots signalés et la masse historique.

### 2.4 Engagement & Communauté
- **Taux de Participation (RSVP)** : Dynamique des bénévoles.
- **Leaderboard d'Impact** : Top contributeurs (actions, kg).
- **Mixité des Sources** : Répartition Citoyen / Associatif / Institutionnel.

## 3. Méthodologie de Calcul (Proxies)

| Indicateur | Formule | Source |
| :--- | :--- | :--- |
| **Eau protégée** | `Nombre de mégots * 500L` | Littérature scientifique standard |
| **CO2 évité** | `Nombre de mégots * 0.0014kg` | Proxy de cycle de vie simplifié |
| **Indice de Tri** | `f(Masse, Mégots, Type)` | Algorithme interne CleanMyMap |

## 4. Seuils de Qualité
- **Critique (< 70%)** : Données insuffisantes pour arbitrage officiel.
- **Crédible (70-85%)** : Utilisable pour pilotage interne.
- **Institutionnel (> 85%)** : Prêt pour diffusion publique et dossiers budgétaires.
