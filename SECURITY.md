# Politique de sécurité CleanMyMap

La sécurité de CleanMyMap est traitée comme un sujet prioritaire.

## Signaler une vulnérabilité

Ne signalez pas de vulnérabilité exploitable via une issue publique.

Pour un signalement responsable, contactez l'équipe de sécurité à l'adresse `security@cleanmymap.fr`, ou utilisez le canal privé prévu par le projet si vous en avez un.

Nous visons les délais suivants :

1. accusé de réception sous 48 h ouvrées ;
2. premier retour sur l'analyse et la portée ;
3. correctif ou atténuation dès que possible selon la criticité.

## Version prise en charge

La branche `main` correspond à la version supportée pour les correctifs de sécurité.
