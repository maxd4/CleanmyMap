# Politique de sécurité CleanMyMap

La sécurité de CleanMyMap est traitée comme un sujet prioritaire.

## Signaler une vulnérabilité

Ne signalez pas une vulnérabilité exploitable via une issue GitHub publique.

Pour un signalement responsable, contactez :

```txt
security@cleanmymap.fr
```

ou utilisez un canal privé explicitement prévu par le projet.

Merci d'inclure si possible :

- surface concernée ;
- conditions de reproduction ;
- impact estimé ;
- preuve minimale ;
- suggestion d'atténuation éventuelle.

N'incluez pas de données personnelles ou de secrets supplémentaires au-delà du strict nécessaire.

## Délais visés

1. accusé de réception sous 48 h ouvrées ;
2. premier retour sur l'analyse et la portée ;
3. correctif ou atténuation selon la criticité.

Ces délais sont des objectifs et non une garantie contractuelle.

## Version prise en charge

La branche `main` correspond à la version supportée pour les correctifs de sécurité.

## Divulgation coordonnée

Merci de laisser au projet un délai raisonnable pour analyser et corriger le problème avant toute divulgation publique.
