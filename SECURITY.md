# Politique de Sécurité - CleanMyMap

La sécurité de la plateforme CleanMyMap est une priorité. Si vous découvrez une vulnérabilité, nous vous remercions de nous en informer de manière responsable.

## Signaler une vulnérabilité

Veuillez ne pas signaler les vulnérabilités de sécurité via les issues publiques de GitHub.

À la place, merci d'envoyer un e-mail à : **security@cleanmymap.fr** (ou utilisez le formulaire de contact admin si disponible).

Nous ferons de notre mieux pour :
1. Accuser réception de votre rapport sous 48h.
2. Proposer un correctif ou une atténuation dans les plus brefs délais.
3. Vous tenir informé de l'avancement de la résolution.

## Versions prises en charge

Seule la dernière version du moteur Next.js dans la branche `main` est officiellement prise en charge pour les correctifs de sécurité.
