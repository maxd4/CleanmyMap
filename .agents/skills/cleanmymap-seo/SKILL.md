# CleanMyMap SEO Skill

## Purpose

Keep search-facing pages structured, explicit, and indexable.

## Use When

- Editing page metadata
- Adding schema.org structured data
- Changing sitemap or canonical behavior

## Core Rules

- Keep titles and descriptions specific.
- Reuse existing JSON-LD patterns when possible.
- Verify that the page is still discoverable and coherent.

## Validation

- Check metadata output.
- Validate structured data when it changes.
- Confirm the page appears in the sitemap if needed.
