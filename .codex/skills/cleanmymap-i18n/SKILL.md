# CleanMyMap i18n Skill

## Purpose

Keep multilingual content, locale routing, and translated UI consistent.

## Use When

- Editing translated strings or locale files
- Touching server locale logic or language-aware routes
- Adding new user-facing text that must exist in FR and EN

## Core Rules

- Keep public-facing text aligned across locales.
- Prefer existing locale helpers and translation patterns.
- Check both language variants when the content is user-facing.

## Validation

- Verify the FR and EN versions.
- Confirm route and metadata behavior in both locales.
