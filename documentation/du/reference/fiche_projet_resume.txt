# Fiche projet — CleanMyMap (version courte)

**Date :** 9 avril 2026  
**Version :** 1.2

---

## 1. Résumé exécutif

CleanMyMap est une plateforme web qui transforme les actions bénévoles de dépollution en informations utiles pour le terrain, les associations et les décideurs publics.

---

## 2. Problème traité

Les résultats des actions citoyennes sont souvent dispersés et difficiles à exploiter. Cela freine la priorisation des zones, la mesure d’impact et la mobilisation des partenaires.

---

## 3. Réponse apportée

- **Action terrain :** savoir où, quand et comment intervenir ;
- **Pilotage associatif :** mieux planifier les opérations ;
- **Culture citoyenne :** diffuser des contenus pédagogiques ;
- **Plaidoyer institutionnel :** objectiver l’impact ;
- **Valorisation des acteurs :** rendre visibles les partenaires engagés.

---

## 4. État réel du projet (09/04/2026)

- Runtime principal : web app Next.js (`apps/web`) ;
- nettoyage legacy terminé (`app.py`, `src/ui/*`, `src/services/*` supprimés) ;
- contrôles techniques validés (lint, tests, build, maintenance checks).

Plan d’exécution :
- E03 à E09 : réalisés ;
- E10 à E20 : en attente (pause).

Décision de gouvernance actée :
- niveau `data-quality` retenu : **complet** (scoring, anomalies, règles avancées, workflows de correction, historique, priorisation).

---

## 5. Livrables disponibles

- application web (dashboard, reports, admin, map, history) ;
- cartographie interactive ;
- API métier ;
- exports CSV/JSON ;
- documentation et scripts de maintenance.

---

## 8. Risques clés

- qualité de données ;
- complexité/délai lié au module data-quality complet ;
- dépendance à des services tiers ;
- charge de travail élevée ;
- financement en cas de forte croissance.

Réponse : contrôles automatisés, modération, priorisation stricte, pilotage budgétaire progressif.

CleanMyMap est techniquement solide et orienté usage réel. Le projet est prêt pour une phase de consolidation fonctionnelle et de diffusion grand public.
