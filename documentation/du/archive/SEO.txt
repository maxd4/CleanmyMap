Analyse mon site et traite ces 5 axes de visibilité web de façon structurée : SEO, AEO, GEO, AIO, SXO.

Objectif :
- identifier les écarts ;
- proposer des améliorations concrètes ;
- prioriser les actions à plus fort impact ;
- préparer une base solide pour la recherche classique, les moteurs de réponse, les interfaces génératives, l’exploitation par IA et l’expérience de recherche.

Définitions de travail :
- SEO = Search Engine Optimization ;
- AEO = Answer Engine Optimization ;
- GEO = Generative Engine Optimization ;
- AIO = AI Optimization ;
- SXO = Search Experience Optimization.

Ce que tu dois faire :
1. Auditer l’existant sur tout le site.
2. Pour chaque acronyme, lister :
   - problèmes détectés ;
   - opportunités ;
   - correctifs concrets ;
   - quick wins ;
   - chantiers structurels.
3. Produire des recommandations actionnables sur :
   - structure HTML ;
   - balisage sémantique ;
   - métadonnées ;
   - schema markup ;
   - hiérarchie Hn ;
   - FAQ et blocs de réponses courtes ;
   - pages piliers et clusters ;
   - maillage interne ;
   - autorité et signaux de confiance ;
   - lisibilité ;
   - citations, sources, données vérifiables ;
   - performance ;
   - mobile ;
   - accessibilité ;
   - UX de recherche et de conversion.
4. Vérifier si le contenu est facilement :
   - indexable par les moteurs ;
   - extractible en réponse directe ;
   - citable par des moteurs génératifs ;
   - compréhensible par des systèmes IA ;
   - agréable et performant pour l’utilisateur.
5. Proposer les modifications exactes à implémenter dans le code et dans le contenu.
6. Quand c’est pertinent, générer directement les patchs.

Contraintes :
- ne fais pas de blabla ;
- sois concret, précis, priorisé ;
- évite les recommandations vagues ;
- conserve le comportement métier existant sauf si une correction est clairement justifiée ;
- n’ajoute pas de complexité inutile.

Format de sortie obligatoire :
- Résumé exécutif
- SEO
- AEO
- GEO
- AIO
- SXO
- Quick wins (faible effort / fort impact)
- Priorité haute
- Priorité moyenne
- Priorité basse
- Fichiers à modifier
- Patches proposés
- Risques / points de vigilance
- Étapes suivantes
