# Atelier DU

Ce fichier synthétise les apports des ateliers DU utiles à CleanMyMap.

## Ce qui a été retenu pour le produit

- privilégier une lecture claire des indicateurs;
- garder la trace des arbitrages techniques;
- séparer les données mesurées, dérivées et estimées;
- documenter les dépendances et les risques de verrouillage;
- relier chaque choix technique à une utilité terrain.

## Ce que cela change pour l'estimateur d'impact

- les courbes doivent rester explicables;
- les calculs doivent être auditables;
- les données manquantes doivent être signalées explicitement;
- les hypothèses doivent rester versionnées;
- les actions de réduction doivent être classées par effet potentiel.
