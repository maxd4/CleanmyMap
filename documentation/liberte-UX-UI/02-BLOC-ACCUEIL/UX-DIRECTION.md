# Direction UX - Section Acceuil

## Mission

La section Acceuil doit orienter rapidement l'utilisateur dans son espace de travail. Ce bloc n'est pas un hub marketing. C'est un point d'entree de pilotage personnel, de reprise d'activite et de contexte utile.

## Rubriques existantes

- `Tableau de bord` -> `/dashboard`
- `Profil & impact` -> `/profil`
- `Explorer`-> `/explore`

## Theme couleur recommande

- Axe chromatique : orange civique, ambre chaud, cuivre doux
- Role : accueil actif, reprise de contexte, chaleur immediate sans tomber dans le marketing
- Fond de bloc : `bg-[linear-gradient(180deg,rgba(58,36,18,0.94),rgba(72,45,20,0.98))]`
- Overlay / glow : `from-orange-500/14 via-amber-500/10 to-transparent`
- Bordure : `border-orange-300/22`
- Hover border : `hover:border-orange-300/42`
- Surface secondaire : `bg-[rgba(97,61,29,0.78)]`
- Ombre : `shadow-[0_24px_56px_-32px_rgba(249,115,22,0.30)]`
- Texte secondaire : `text-white/78`
- Chips / badges : `bg-orange-500/14 text-orange-100 border-orange-200/18`
- Regle stricte : aucun blanc ni noir sur les surfaces, bordures, overlays ou ombres. Reserve au texte uniquement.

## Direction UX

- Priorite a la reprise de contexte: "ou j'en suis", "quoi faire ensuite", "quel est mon perimetre".
- Lecture immediate, sans surcharge visuelle ni grands effets decoratifs.
- Les informations principales doivent etre visibles sans scroller sur desktop.
- Les CTA doivent aider a repartir vers l'action ou vers le suivi personnel.

## Regles d'interface

- Utiliser une hierarchie tres claire: statut, progression, prochaines actions.
- Favoriser des blocs courts, denses, comparables.
- Eviter les longs textes introductifs.
- Ne pas faire du bloc Home un doublon des autres blocs.

## Signaux de reussite

- L'utilisateur comprend son statut en moins de 5 secondes.
- Les prochaines actions sont visibles sans chercher.
- Le bloc sert d'entree utile, pas de page de presentation.

## A eviter

- Hero marketing
- cartes decoratives surdimensionnees
- tunnels d'onboarding inutiles
- repetition de navigation deja visible dans le header
