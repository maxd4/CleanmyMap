Règles globales

Le poster termine par les règles de composition qui évitent les dérives visuelles: rayons, hiérarchie des fonds, contraste, espacements et états.

Lecture rapide
essentiel / stable / accessible
Rayons

8 / 16 / 24 / full

Le système garde une progression nette entre tags, champs, cartes et sections.

  • 8 px pour les micros-badges.
  • 16 px pour les controls et champs.
  • 24 px pour les cards majeures.
Fonds

Hiérarchie claire

Canvas clair, surfaces élevées blanches, surfaces secondaires grises et glassmorphism retenu.

  • Le fond ne doit pas écraser les cartes.
  • Le glissement entre niveaux reste subtil.
Contraste

Lisibilité d’abord

Le texte principal reste sombre sur fond clair et les états actifs sont clairement séparés.

  • Focus visible sur clavier.
  • Hover lisible, jamais décoratif.
Espacements

Grille régulière

La base 8 px pilote l’ensemble. Les sections respirent, les micro-composants restent compacts.

  • 12-14 px pour les labels.
  • 18-24 px pour les panneaux et cards.
