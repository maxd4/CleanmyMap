# Prospects CleanMyMap — acteurs actifs de dépollution en France

Date de génération : 2026-06-30  
Périmètre temporel : acteurs avec une preuve publique d'action, d'événement, de mission ou d'activité de ramassage entre 2024-06-30 et 2026-06-30, ou avec un agenda 2026 actif.

## Objectif

Cette fiche recense des associations, collectifs, collectivités, plateformes et entreprises susceptibles de bénéficier de CleanMyMap : carte des zones nettoyées ou signalées, formulaire terrain, bilan d'impact, historique local, suivi des déchets, photos de preuve, valorisation des bénévoles et coordination entre acteurs.

Le fichier est organisé par ville, avec Paris en premier. Dans chaque ville, les lignes sont classées par volume connu, fréquence ou potentiel d'activité : réseau national ou gros volume d'abord, actions récurrentes ensuite, puis opérations ponctuelles récentes.

## Limites de lecture

Ce document est une base de prospection qualifiée, pas un recensement exhaustif officiel. Les volumes ne sont pas toujours comparables : certains acteurs publient un nombre total d'actions, d'autres seulement une mission ponctuelle ou une fréquence. Quand le nombre exact d'actions récentes n'est pas disponible, la colonne "Volume / fréquence" indique le niveau le plus solide trouvé publiquement.

Critère appliqué : suppression ou mise de côté des structures sans preuve claire de ramassage dans les deux dernières années, sauf réseau national structurant ou acteur stratégique dont l'activité récente est confirmée.

## Priorités rapides

1. Partenariats données / méthodologie : MerTerre, Surfrider / Retrace, Wings of the Ocean, World Cleanup Day France, CleanWalker, Project Rescue Ocean.
2. Tests Île-de-France : CleanWalker Paris, GreenMinded, Surfrider Paris, The Clean Project, Association 17 et nous, PikPik Environnement, Paris La Défense, Rueil-Malmaison, Chaville.
3. Déploiement littoral : Marseille, Morbihan, Biarritz, Montpellier, Sète, La Ciotat, Côte d'Azur.
4. Prospection collectivités : Rueil-Malmaison, Chaville, Cenon, Nantes Métropole, Saint-Avertin, Surgères, Douai, Strasbourg, La Ciotat.

## Méthode recommandée pour rendre la base vraiment exhaustive

Pour atteindre une base nationale beaucoup plus complète, il faut passer d'une fiche Markdown à un tableau structuré, puis importer les résultats par lots.

Champs recommandés :

| Champ | Exemple |
|---|---|
| id_structure | `cleanwalker-paris` |
| nom | `CleanWalker Paris` |
| ville_principale | `Paris` |
| zone_intervention | `Paris / Île-de-France` |
| type_structure | `association`, `collectivité`, `entreprise`, `collectif`, `plateforme` |
| preuve_action_recente | `Mission JeVeuxAider 2025`, `agenda 2026`, `bilan 2025` |
| nombre_actions_confirme | nombre uniquement si source fiable |
| fréquence | `mensuel`, `annuel`, `ponctuel`, `réseau national` |
| source_chiffre | URL |
| source_action_recente | URL |
| contact_public | page contact / formulaire / email public |
| intérêt_cleanmymap | `carte`, `formulaire`, `rapport`, `données`, `partenaires`, `pédagogie` |
| priorité | `critique`, `forte`, `moyenne`, `faible`, `à vérifier` |
| statut_contact | `non contacté`, `mail envoyé`, `réponse reçue`, `refus`, `partenaire` |

Sources à exploiter en priorité :

- JeVeuxAider.gouv.fr : recherche par activité "ramassage de déchets", puis extraction ville, structure, date, lien.
- J'agis pour la nature : extraction des activités World Cleanup Day, cleanups et nettoyages de plage.
- World Cleanup Day France : bilans régionaux, Mégothon, missions, ambassadeurs et partenaires.
- Zéro Déchet Sauvage / MerTerre : structures mobilisées, ramassages, tonnages, caractérisation.
- Retrace / Surfrider : collectes et données de terrain.
- CleanWalker : antennes locales et événements.
- Pages de collectivités : World Cleanup Day local, opérations mégots, brigades vertes, nettoyages citoyens.
