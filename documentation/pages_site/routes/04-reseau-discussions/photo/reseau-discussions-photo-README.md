# Réseau & Discussions - Photo du bloc

Ce dossier centralise les captures du bloc `Réseau & Discussions`.

## Règles

- Captures en `WebP` uniquement.
- Pas de sous-dossier photo par route.
- Nommer les fichiers avec la route, le libellé lisible et la date de capture.
- Conserver une seule source d'images pour les routes et sections du bloc, notamment `community`, `feedback`, `actors`, `annuaire`, `messagerie`, `open-data`, `funding`, `trash-spotter`, `partners/dashboard` et `partners/onboarding`.

## Convention conseillée

- `04-reseau-discussions__sections-community__communaute__YYYY-MM-DD.webp`
- `04-reseau-discussions__sections-feedback__retours-qualite__YYYY-MM-DD.webp`
- `04-reseau-discussions__messagerie__messagerie__YYYY-MM-DD.webp`

## Notes

- Si une capture mobile est nécessaire, l'indiquer dans le nom du fichier plutôt que créer un autre dossier.
- Ce dossier ne contient pas la documentation métier des pages, uniquement les images du bloc.
