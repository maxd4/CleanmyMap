# Photo du bloc

Ce dossier est l'unique emplacement des captures du bloc `Réseau & Discussions`.

## Règles

- Captures en `WebP` uniquement.
- Pas de sous-dossier photo par route.
- Nommer les fichiers avec la route, le libellé lisible et la date de capture.
- Conserver une seule source d'images pour `community`, `feedback`, `messagerie`, `open-data`, `partners/dashboard` et `partners/onboarding`.

## Convention conseillée

- `04-reseau-discussions__community__communaute__YYYY-MM-DD.webp`
- `04-reseau-discussions__feedback__retours-qualite__YYYY-MM-DD.webp`
- `04-reseau-discussions__messagerie__messagerie__YYYY-MM-DD.webp`

## Notes

- Si une capture mobile est nécessaire, l'indiquer dans le nom du fichier plutôt que créer un autre dossier.
- Ce dossier ne contient pas la documentation métier des pages, uniquement les images du bloc.
