# Mode École — Objectifs non pertinents

But : conserver la mémoire locale des idées explicitement écartées.

## Liste des objectifs écartés

### Créer un compte obligatoire pour chaque élève

Écarté tant que le modèle produit reste un atelier collectif sans compte élève.

Motifs :

- friction ;
- données personnelles inutiles ;
- complexité scolaire ;
- contradiction avec la promesse actuelle.

### Ajouter un classement public des élèves

Écarté.

Motifs :

- compétition inutile ;
- risque de détournement pédagogique ;
- incohérence avec la gamification non compétitive.

### Ajouter des animations lourdes pour « rendre la page fun »

Écarté par défaut.

Motifs :

- performance ;
- distraction ;
- accessibilité ;
- aucun gain pédagogique démontré.

## Règle de mémoire

Ne pas reproposer une idée écartée sans élément nouveau.

Si le contexte change, documenter :

- nouvel élément ;
- raison de reconsidérer ;
- décision mise à jour.
