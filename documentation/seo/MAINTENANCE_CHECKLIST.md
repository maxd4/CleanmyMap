- [ ] Vérifier Google Search Console pour erreurs d'indexation

- [ ] Vérifier les performances Core Web Vitals

## Mensuel
- [ ] Ajouter les nouvelles pages au sitemap
- [ ] Vérifier les мета descriptions (150-160 caractères)
- [ ] Tester les schemas JSON-LD avec Google Rich Results
- [ ] Mettre à jour les mots-clés si nouvelles fonctionnalités
- [ ] Vérifier les redirections (HTTP → HTTPS, www → sans www)

## Trimestriel
- [ ] Ajouter 2-3 nouvelles questions FAQ
- [ ] Créer un nouveau HowTo schema si nouveau feature
- [ ] Audit backlinks - retirer liens toxiques
- [ ] Analyser les positions mots-clés principaux

## Annuel
- [ ] Refonte complète des мета données
- [ ] Audit technique complet (Vitesse, Mobile, Indexation)
- [ ] Mise à jour du robots.txt (nouveauxAI bots)
- [ ] Revoir la stratégie de mots-clés
- [ ] Vérifier les redirections permanentes (301)
- [ ] Créer un rapport annuel SEO

## Signaux d'alerte

### Urgent - Action immédiate
- Pages non indexées (Check sitemap.xml)
- Erreurs 5xx server
- Contenu dupliqué détecté

### Important - À traiter sous 1 semaine
- Baisse de positionnement significative
- Erreurs 404 (liens cassés)
- Temps de chargement > 3s mobile

### Normal - À traiter ce mois
- Images sans alt texte
- Liens internes cassés
- Métadonnées manquantes sur nouvelles pages
